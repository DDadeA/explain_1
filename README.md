## JAVA로 로고 만들기
> 유년 시절 프로그래밍에 관심이 많았던 탓에 기초적인 프로그래밍 능력이 있었음. 수학 교과에서 진행한 그래프로 로고 만들기 활동에서 이러한 특기를 활용하여 로고를 컴퓨터로 작성, 발표 영상을 학교 밴드에 게시함.
